## Descripcion
**QR Scanner** es un programa simple que utiliza la libreria tkinter para su GUI, la libreria CV2 para acceder a la camara y FileDialog para extraer archivos, el programa permite
escanear un codigo QR atraves de una imagen o directamente atraves de la camara y ofrece la url al detectarla.

| Libreria | Uso |
|---|---|
| Tkinter | GUI |
| CV2 | Acceso a camara |
| Filedialog(tkinter) | Acceder a imagenes |
